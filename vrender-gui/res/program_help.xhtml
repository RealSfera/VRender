Справка по программе VRender

    Вот некоторые математические понятия:
    Скалярное поле: если каждой точке M некоторой области некоторого пространства поставлено в соответствие некоторое (обычно — действительное) число u, то говорят, что в этой области задано скалярное поле. Другими словами, скалярное поле — это функция, отображающая Rn в R (скалярная функция точки пространства). [Взято с Wikipedia]
    Изо-уровень - некоторое постоянное значение, которое находится в скалярном поле, и которое образует поверхность (если скалярное поле - трёхмерное). Таким образом, в каждой точке поверхности находится одно и то же значение (изо-уровень).

    VRender - построитель и визуализатор трёхмерного скалярного поля.
    Построение скалярного поля производится вводом требуемой функции в поле Функция построения, выбором размера скалярного поля (по-умолчанию, установлен 128) и нажатии кнопки НАЧАТЬ ПОСТРОЕНИЕ.

    Визуализация скалярного поля производится автоматически в режиме реального времени, любые изменения настроек визуализации автоматически применяются.

    В сцене с объектом присутствует один источник света, его параметры, а также материал объекта, можно задавать на вкладке Освещение и цвет. Объект можно вращать при помощи мыши (при помощи виртуального трэкбола). С помощью колёсика мыши можно приближать\отдалять (zoom). Есть возможность экспорта полученного объекта в файл формата Wavefront (.obj), при этом экспортируется объект полигонизированный с текущим изо-уровнем.

    На вкладке Базовые можно настроить: Размер поля полигонизации, влияющее на детализацию объекта, Изо-уровень задаёт значение, из скалярного поля, на основе которого производится полигонизация поверхности объекта, также можно установить Постепенное изменение - это пошаговое изменение изо-уровня, где можно задать значение от, до и шаг изменения (шаг в кадр визуализации).

    На вкладке Освещение и цвет можно устанавливать цвет внутренней и внешней поверхности объекта, цвет источника света и цвет блика, причём сделать это можно либо вручную устанавливая состовляющие цвета (от 0 до 255 для каждой состовляющей) или нажав на кнопку =, можно установить цвет, используя диалог выбора цвета.

    Сила блика - это то, насколько блик будет более сжатым (т.е. сильным, сконцетрированным).
    Коэффициенты освещения - множители, которые задают усиление (или уменьшение) влияния соответствующей компоненты света.   
    Гамма - коррекция яркости.

    Используя Угол расположения источника света можно задать угол вращения источника относительно позиции смотрителя (камеры); угол задаёт вращение по часовой стрелки. При помощи Анимация вращения источника можно задать шаг, на который будет изменяться угол вращения источника (по часовой стрелки; если шаг отрицательный - против) за один кадр визуализации.
